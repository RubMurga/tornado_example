# Python example Tornado

This is a Tornado example. 

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install -r requirements.txt
```

## Usage

```bash
python server.py
```

Go to localhost:4200
